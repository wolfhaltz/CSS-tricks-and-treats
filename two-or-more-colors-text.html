Is possible two colors in
one text using only CSS?

Yes! It is! First, you need do the class and establish the background, changing what you want, just like:

.two-colors-abrupt{
background: linear-gradient(to right, black 50%, darkred_02 0);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
}

If you use 0 at the end, the color will result something abrupt, just like:
Dracula

But, if you want something less abrupt, just like this one:
Dracula
So you need to use %:
.two-colors-smooth{
background: linear-gradient(to right, black 40%, darkred 75%);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
}
